# @minecraft-types/jdk.graal.compiler

TypeScript definitions for `jdk.graal.compiler` from JDK `22.0.2\`.

## Install

Install the package via npm or pnpm:

```bash
npm install @minecraft-types/jdk.graal.compiler
# or
pnpm add @minecraft-types/jdk.graal.compiler
```

Then, in your `tsconfig.json`, add the following to the `compilerOptions` section:

```json
{
  "compilerOptions": {
    "typeRoots": [
      "node_modules/@minecraft-types/",
      "node_modules/@types"
    ]
  }
}
```
